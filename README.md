# Temps Homebrew Tap

Official Homebrew tap for gotempsh tools.

## Installation

First, tap the repository:

```bash
brew tap gotempsh/tap
```

Then install temps:

```bash
brew install temps
```

Or install directly:

```bash
brew install gotempsh/tap/temps
```

## Available Formulae

- **temps** - Temporary file and directory management tool

## Updating

To update to the latest version:

```bash
brew update
brew upgrade temps
```

## Uninstalling

```bash
brew uninstall temps
brew untap gotempsh/tap
```

## Development

To add or update formulae in this tap, please submit a pull request to this repository.
